# Secured-QR
>
Encrypted QR codes are QR codes that everyone cannot scan and access. 
They are not too common

## Installation

>Use the package manager [pip](https://pip.pypa.io/en/stable/) to Use Secured-QR
```bash
pip install tk
pip install qrcode
pip install pyzbar
pip install Pillow
pip install opencv-python
pip install image
```
